Fala pessoal, eu sou o Eduardo! :wave:😊

Atualmente atuo como **AI Software Engineer**, com foco em construir aplicações inteligentes usando **LLMs**, **Agents** e **RAG**. Minhas principais tecnologias são **Python**, **LangChain**, **LangGraph**, **OpenAI**, **Anthropic**, **Vector Databases**, **FastAPI**, **Docker**, **GCP** e **Terraform**.

Meu background é em **Engenharia de Software**, mas sempre transitei pelo mundo de **dados** — passando por Engenharia de Dados e Analytics Engineering até chegar em IA aplicada. Essa jornada me dá uma visão completa: da construção de pipelines e arquiteturas de dados até o desenvolvimento de agentes autônomos, sistemas de RAG e APIs em produção.

Sou apaixonado por construir soluções que combinam **engenharia de software de qualidade** com **inteligência artificial aplicada**, sempre buscando seguir as boas práticas de arquitetura, testes e observabilidade.
